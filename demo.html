Demo for the search & select input

Features :

  1. You could try click the DOWN arrow to act a select feature and try filter the options by input some key words.
  2. UP and DOWN arrow at keyboard are available to navigate the items in the filtered list.
  3. a green border will show after you choose one of the items either by ENTER key or by click.
  4. sure you could clear you selection by clicking the BIG cross align right.
  5. you could open the debug mode by setting the debug to true to keep track of the value you'll post.
